This project is a simple data analytics dashboard built using Streamlit, Python, and pandas. The dashboard analyzes customer data from a European bank dataset to understand and visualize customer churn patterns.

The application loads customer data from a CSV file and provides an interactive dashboard that displays important churn metrics and visualizations.

🔹 Features

Displays the overall customer churn rate

Shows geography-wise churn comparison

Provides a visual bar chart for churn analysis

Helps identify which regions have higher customer churn

🔹 How It Works

The dataset is loaded using pandas.

The churn rate is calculated using the Exited column.

Data is grouped by Geography to analyze churn trends.

Streamlit displays the results through metrics and charts in a web dashboard.

🔹 Technologies Used

Python

Streamlit

Pandas

🔹 Purpose

The goal of this project is to demonstrate how data analytics and visualization can help businesses understand customer behavior and identify churn patterns for better decision-making.
